criar uma pasta vagrant-lab (criei dentro da pasta do curso)

colocar o arquivo com os recursos do site e colocar nesta pasta

acessar a pasta vagrant-lab via powershell

(OBS: no windows o vagrant não roda com o Hyper-V ativado)

colocar "vagrant init"

arquivo VagrantFile foi criado

colocar:

config.vm.box = "centos/7"

descomentar:

config.vm.network

colocar depois do host=8080:

host_ip = "127.0.0.1

descomentar o provision:

config.vm.provision "shell", path: "provision.sh"

criar na pasta um arquivo sh:

provision.sh

no arquivo, colocamos como comentario que este é um bash:

#!/user/bin/env bash
echo "Installing Apache and setting it up..."
yum install -y httpd >/dev/null 2>&1 # instalação do apache
cp -r /vagrant/html/* /var/www/html/ # copiando tudo da pasta html para o diretorio padrão que disponibiliza paginas HTTP
service httpd start # subir o apache

para start:

no terminal:

vagrant up

no terminal:

vagrant status # para saber se a maquina subiu ou não
